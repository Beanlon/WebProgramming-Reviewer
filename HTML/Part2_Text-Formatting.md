# Text Formatting

Being able to do text formatting allows for better paragraph creations for better visually engaging websites

**Examples**

|Format             |Start tag| End tag |
|-------------------|---------|---------|
|   Bold            | `<b>`   | `</b>`  |
|   Underline       | `<u>`   | `</u>`  |
|   Italisize        | `<i>`   | `</i>`  |             
|   Struckthrough   | `<s>`   | `</s>`  |           
|   Subscript       | `<sub>` | `</sub>`  |             
|   Superscript     | `<sup>` | `</sup>`  |             

## Bold (b)
Using bold allows for thicker fonts when indicating text which can be done by `<b>` as the start tag and `</b>` with an end tag

#### Example

```https
<p>This is <b>Bold</b> text.</p>
```

## Underline (u)
Underlined are mainly used to show lines underneath the text which is useful when highlighting certain description using `<u>` and `</u>`.

#### Example

```https
<p>This is <u>underline</u> text.</p>
```
## Italisize (i)
Italisizing crreates for contrasting of style between texts which uses `<i>` and `</i>` as their tags.

#### Example

```https
<p>This is <i>italisize</i> text.</p>
```
## Struckthrough (s)
Struckthrough is a line that crosses through the text which uses `<s>` and `</s>` as their tags.

#### Example

```https
<p>This is <s>strucktrhough</s> text.</p>
```
## Subscript (sub)
Subscript is a rare usage of formatting which are small characters that are on the lower side of the text mostly used in chemical formulas and numbering like H20 etc. it uses `<sub>` and `</sup>` as their tags.

#### Example

```https
<p>This is <sub>subscript</sub> text.</p>
```
## Superscipt (sup)
Superscript is the opposite of subscipt as it is shown on top of values like exponents in numbers it uses `<sup>` and `</sup>` as their tags.

#### Example

```https
<p>This is <s>supercript</sup> text.</p>
```
