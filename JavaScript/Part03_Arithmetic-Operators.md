# Arithmetic Operators
These are simply the basic operations we used to know

* `+` - Addition
* `-` - Subtraction
* `*` - Multiplication
* `/` - Division

#### Example

```https
let sum = 1 + 2;
let output = console.log(sum); //output is 3
```
So goes for the other operators

## Extra operators
Here are some extra operators needed for more operations
* `%` - modulo
* `**` - exponent
* `--` - decrement
* `++` - increment
* `Math.floor()` - Shows the value nearest to the lower side of the value
* `Math.ceil()` - Shows the value nearest to the higher side of the value
* `Math.round()` - Shows the rounded off value
* `Math.random()` - Shows a random number


#### Example

```https
let num = 4;
num++; //increment
console.log(num); //Output is 5;

let num = 4;
num--; //decrement
console.log(num); //Output is 3;

let num = Math.floor(4.3);
console.log(num); //Output is 3;

let num = Math.ceil(4.6);
console.log(num); //Output is 5;

let num = Math.round(4.8);
console.log(num); //Output is 5;

let random - Math.random();
console.log(Math.random) // Output could be any number;
```
